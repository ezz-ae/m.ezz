# **App Name**: Ezz.AE Showcase

## Core Features:

- Architecture Portfolio Display: Showcase Mahmoud Ezz's architectural projects with high-quality images and descriptions.
- Project Filtering and Search: Allow users to easily filter and search projects based on categories, keywords, or project types.
- AI-Powered Design Insights: Use AI to analyze Mahmoud Ezz's architectural designs and generate design insights to highlight key architectural elements and concepts.  The AI acts as a tool.
- Contact Form: Provide a contact form for potential clients or collaborators to get in touch.
- About Section: Include an 'About' section with information about Mahmoud Ezz, his architectural philosophy, and design approach.
- Blog/News Section: Showcase project news or personal views

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to evoke a sense of sophistication and grounding in architectural design.
- Background color: Light grey (#FAFAFA) for a clean and modern backdrop, subtly complementing the beige.
- Accent color: Soft brown (#A0522D), providing a contrast against the primary, evoking earth tones found in architectural elements.
- Body font: 'PT Sans', a versatile humanist sans-serif, for body text, striking a balance between modern clarity and subtle personality.
- Headline font: 'Playfair', a modern serif, for headlines, adding an element of refined elegance. If longer text is anticipated, use 'PT Sans' for body.
- Use minimalist, line-based icons to represent project categories and navigation elements.
- Employ a grid-based layout with generous whitespace to create a visually balanced and uncluttered presentation of architectural projects.
- Incorporate subtle, elegant transitions and animations when navigating between projects or sections of the website to enhance user engagement.
